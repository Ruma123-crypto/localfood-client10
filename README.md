# Local Food Lovers Network

Live Site: [https://unique-cranachan-1ac663.netlify.app/](https://your-live-site-link.com)

## Project Description
**Local Food Lovers Network** is a full-stack MERN application that connects food enthusiasts who love exploring local restaurants, street food, or home-cooked meals. Users can share their food experiences, post reviews with photos, and discover what others are enjoying nearby. It’s a community-driven platform celebrating great food, honest opinions, and local flavor.

## Features
- Explore and discover local restaurants, street foods, and homemade dishes.
- Users can post reviews with photos and ratings for any food item.
- View top-rated reviews dynamically on the homepage.
- Add reviews, edit or delete your own reviews (CRUD functionality).
- Favorite system to save your favorite reviews for easy access.
- Search functionality to find foods using MongoDB queries.
- Authentication via Firebase (Email/Password + Google).
- Responsive design for mobile, tablet, and desktop devices.
- Clean, unique UI design with hero sliders, featured reviews, and additional sections.

## Main Pages
- **Homepage**: Hero section, featured reviews, and dynamic sections.
- **All Reviews**: List of all reviews with search and favorite options.
- **Add Review**: Protected page for logged-in users to submit reviews.
- **My Reviews**: Protected page showing all reviews added by the logged-in user with edit/delete options.
- **My Favorites**: Shows all favorite reviews of the currently logged-in user.
- **Authentication Pages**: Login & Registration with Google sign-in option.
- **404 Page**: Fun and user-friendly 404 page with a "Back to Home" button.

## UI & Layout
- Navbar with website logo, navigation links, and user dropdown menu.
- Footer with website info, links, and social media icons.
- Grid layout for consistent card and image sizes.
- Consistent headings, paragraph spacing, and button styles.
- Loading spinners and skeletons while fetching data.
- Fully responsive for all devices.

## Technologies Used
- **Frontend**: React.js, React Router, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: Firebase Authentication (Email/Password + Google)
- **Hosting**: Client-side (Netlify/Surge/Firebase), Server-side (Vercel)
- Optional: Tanstack Query for data fetching, React Hook Form for form handling
